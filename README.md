# Spotify Streaming Analysis

Built a centralised dataset and real-time dashboard to reduce reporting time by 20 hours/week.

## Problem
Streaming data was siloed across multiple platforms, requiring 20+ hours/week of manual reporting with no real-time visibility into release performance.

## Approach
- Consolidated data into a centralised dataset
- Built an interactive Excel dashboard for real-time tracking
- Conducted root-cause analysis on underperforming releases to identify drop-off points

## Tools
- Excel (pivot tables, slicers, automated charts)

## Results
- Reduced reporting time by 20 hours/week
- Informed strategies contributing to 100M+ impressions and 10x daily stream growth
- Supported release reaching Billboard Hot 100

## Repository Structure
- `data/` — sanitized sample datasets
- `docs/` — dashboard screenshots
- `analysis/` — Excel workbook with full analysis
